# 📡 MediaMTX Streaming System

## 🚀 Overview
Low-latency RTSP streaming system designed for multi-camera surveillance environments.  
Supports real-time video streaming with minimal delay and high reliability.

---

## ⚙️ Features
- Supports **10+ camera streams simultaneously**
- Achieves **2–3 seconds startup latency**
- Minimal frame loss during streaming
- Real-time control via backend APIs
- Scalable architecture for surveillance systems

---

## 🛠️ Tech Stack
- Python
- MediaMTX
- RTSP Protocol
- FastAPI (integration)
- Linux-based deployment

## ▶️ How to Run

1. Install MediaMTX
2. Use the provided configuration file
3. Run:
   ./mediamtx mediamtx_config.yml

4. Access RTSP stream:
   rtsp://localhost:8554/camera1
